this code gets the longest prefix of a phone number and where there are no prefix it saves an empty string in the array and returns the original list with phone numbers

example

phone numbers =['+44829', '+87249', '+23478','+88047', '+123455', '+144899', '+880999'] 
prefix = ['+44', '+880', '+234', '+144', '+1']

output = ['+44', ' ', '+234', '+880', '+1', '+144', '+880']
